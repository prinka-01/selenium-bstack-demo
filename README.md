# Selenium Automation Framework – BrowserStack Demo

## 📌 Overview

This project is a Selenium automation framework built using Java and TestNG.
It automates key user flows of an e-commerce demo application.

## 🧰 Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven

## 🚀 Features

* Page Object Model (POM)
* End-to-End Test Scenarios
* Reusable Utilities

## ▶️ How to Run

Option 1: Run all tests using mvn test

Option 2: Run using TestNG XML
